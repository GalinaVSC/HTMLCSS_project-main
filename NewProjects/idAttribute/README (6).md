## Key Points
- Uniqueness: The `id` must be unique within a document.
- Targeting Specific Elements: Use `#id_value` in CSS to style an element with a specific `id`. 
- Hover Effects: You can combine pseudo-classes (e.g., `:hover` ) with  selectors for interactive styles.

To center the button we first turn it into a block element:

`display: block;` /* Make the button a block-level element */\
`margin: 0 auto;` /* Center the button horizontally */
- margin-top и margin-bottom: 0 пикселей (т.е. нет верхнего и нижнего отступов)
- margin-left и margin-right: auto (т.е. отступы слева и справа будут автоматически рассчитываться браузером)
